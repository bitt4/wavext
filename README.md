# wavext
Extract wav files from other files

## Installation
```bash
git clone https://github.com/bitt4/wavext
cd wavext
make
./wavext
```

## Usage
Use `./wavext FILE` to extract wave files from FILE.

## Options
| Option                  | Description                |
|-------------------------+----------------------------|
| `-o`, `--output=PREFIX` | Specify output file prefix |
| `-h`, `--help`          | Display help               |
